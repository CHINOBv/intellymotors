# Prueba Tecnica

Inicio del BackEnd

```

cd server

npm install o yarn

npm start o yarn start

```

Inicio del FrontEnd

```
cd client

npm install o yarn

npm start o yarn start
```

## Tecnologias

BackEnd:
NodeJS 14.15.1,
Express, Puppeteer

FrontEnd: ReactJS
